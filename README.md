# Projeto de Agendamento

Sistema simples de agendamento de serviços desenvolvido em JavaScript.

## Funcionalidades
- Criar agendamento
- Cancelar agendamento
- Exibir informações

## Estrutura
O sistema utiliza Programação Orientada a Objetos com:
- Classes
- Herança
- Relacionamentos entre objetos
